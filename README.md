# OpenXava Spring Boot Example

This repository contains a sample project that integrates OpenXava 7.7 with Spring Boot 2.7.

The goal of this project is to serve as a practical reference for configuring and running an OpenXava application inside a Spring Boot-based project.

For detailed instructions on how to perform this integration in your own project, see the official documentation:

https://openxava.org/OpenXavaDoc/docs/springboot_en.html
